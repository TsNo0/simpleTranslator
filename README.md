Simple Translator
Волков Никита M3106

Описание
Плагин для VS Code, позволяющий мгновенно переводить выделенный текст в Google Translate.

Функционал
Работает по горячей клавише Ctrl+T.
Автоматически определяет направление перевода (RU ↔ EN).
Удаляет символы форматирования перед переводом.
Установка:
Клонировать репозиторий.
npm install
F5 для запуска отладки.
